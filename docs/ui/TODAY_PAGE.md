# Today Page Style Guide

## Purpose

Today is the default learning workspace. It is the first screen users see when they open JLS/JTEP.

## Principle

Today follows: One Screen, One Decision.

The page should help the learner understand the first step within a few seconds. It should not become a crowded dashboard.

## Learning Flow

Today

→ 今日焦點

→ 開始今日學習

→ 今日歷程

→ 今日建議

## Content Rules

- Today shows only one 今日焦點.
- Today shows exactly one 今日歷程.
- Today shows at most one 今日建議.
- Today does not include a separate reminder card because it overlaps with 今日建議.
- Navigation labels use English, such as Today.
- Content section labels use Traditional Chinese.
- Today does not include leaderboard, badge, advertisement, or gamification.
- Today does not show lifetime analytics. Lifetime totals belong to future Analytics.

## Tone

Today should feel calm, focused, and learning-oriented. It should reduce choice overload and make the next practice action obvious.
