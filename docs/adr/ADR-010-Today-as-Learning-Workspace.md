# ADR-010: Today as Learning Workspace

## Status

Accepted

## Context

JTEP has Foundation, Memory, Insight, and Knowledge layers. Users still need a first screen that tells them what matters today.

## Decision

Make Today the default landing page. Today presents one focus, one learning journey, and one recommendation.

## Consequences

The app becomes a daily learning workspace rather than a generic question bank. Home remains available at `/home`, but `/` now represents the daily learning decision.
